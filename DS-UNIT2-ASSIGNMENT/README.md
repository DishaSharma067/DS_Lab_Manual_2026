# 📘 Data Structures - Unit 2 Assignment

This repository contains the completed laboratory assignments and implementations for **Unit 2 of the Data Structures curriculum**.

The focus of this unit is on **foundational linear data structures**, emphasizing:

* Dynamic memory allocation
* Pointer-based traversals
* Abstract Data Type (ADT) design using Python

---

## 📂 Repository File Structure

The project has been architecturally segmented into logical components for clarity and modularity.

```
DSA_SectionA_Unit2_Assignment/
│
├── Arrays/
│   └── dynamicArray.py        # Implementation of a dynamic array with resizing logic
│
├── Linked_Lists/
│   ├── SLL.py                # Singly Linked List with insertion, deletion, traversal
│   └── DLL.py                # Doubly Linked List with bidirectional traversal
│
├── Stacks_and_Queues/
│   ├── StackSLL.py          # Stack implemented using Linked List (O(1) operations)
│   ├── QueueSLL.py          # Queue implemented using Linked List
│   └── parenthesis.py       # Parenthesis validation using Stack
│
└── README.md                # Project documentation (this file)
```

## ⚙️ Environment & Execution

All source code files are written in **standard Python 3** and do not require any external third-party dependencies.

### ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone <your-repo-link>
   ```

2. Navigate to the project folder:

   ```bash
   cd DSA_SectionA_Unit2_Assignment
   ```

3. Run any file:

   ```bash
   python filename.py
   ```
