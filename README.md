# 📘 DS Lab Manual 2026 – ETCCDS202

This repository contains complete implementation of the Data Structures Lab Manual for the course **ETCCDS202** (B.Tech – School of Engineering & Technology).

## 📂 Structure

### 🔹 Unit 1 – Foundations & Algorithmic Analysis
- Stack ADT implementation
- Complexity drill (O(1), O(n), O(n²), O(log n))
- Recursive Factorial
- Fibonacci (Naive vs Memoized)
- Tower of Hanoi
- Recursive Binary Search
- Assignment summary with complexity explanation

### 🔹 Unit 2 – Linear Data Structures
- Arrays (1D & 2D operations)
- Dynamic Array simulation
- Singly & Doubly Linked List
- Stack using Linked List
- Queue using Linked List
- Parentheses Checker

### 🔹 Unit 3 – Sorting Algorithms
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Benchmark comparison

### 🔹 Unit 4 – Non-Linear Data Structures
- Binary Search Tree (Insert/Search/Delete)
- Graph (Adjacency List)
- BFS & DFS
- Hash Table (Separate Chaining)
- Trie
- Bloom Filter (Concept Demo)

### 🚀 Final Capstone
Social Network Explorer (SNE) – Graph + Hashing + BFS/DFS integration project.

## 🎯 Learning Outcomes
- Strong understanding of data structure implementation
- Clear complexity analysis (Time & Space)
- Recursive thinking and divide & conquer approach
- Industry-oriented problem-solving skills
